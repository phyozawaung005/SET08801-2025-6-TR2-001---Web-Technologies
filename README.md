# **WORDFALL – TYPING SPEED GAME** 
**WordFall** is a browser-based typing game designed to improve typing speed, accuracy, and reaction time in a fun and interactive way. Words fall from the top of the screen, and the player must type them correctly before they reach the bottom.

---

## **FEATURES**

- **Kid Mode** – Easy words with colorful theme  
- **Adult Mode** – Moderate difficulty words  
- **Gamer Mode** – Dark theme with increased challenge  
- **60-second game timer**  
- **Lives system**  
- **Score tracking**  
- **Words Per Minute (WPM) calculation**  
- **Accuracy percentage calculation**  
- **Restart game functionality**

---

## **TECHNOLOGIES USED**

- **HTML5**  
- **CSS3**  
- **JavaScript**  
- **Browser Local Storage API**

---

## **HOW TO PLAY**

1. Select your preferred game mode.  
2. Words will fall from the top of the screen.  
3. Type the word correctly before it reaches the bottom.  
4. You lose a life if a word reaches the bottom.  
5. The game ends after 60 seconds or when all lives are lost.  
6. Your score and performance statistics will be displayed at the end.

---

## **DATA STORAGE**

The game uses the browser’s **Local Storage API** to store:

- High scores per mode  
- Performance statistics  
- Selected game mode  

Data is stored in structured JSON format using key-value pairs.
